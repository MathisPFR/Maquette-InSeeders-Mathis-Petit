# Maquette Melody Music (Mathis Petit)

## Introduction

Bonjour, je suis Mathis Petit. Dans ce fichier README, je vais vous expliquer la démarche du projet Melody Music, en partageant mes réussites, mes échecs, et les solutions trouvées au cours de ces derniers jours. Ce projet vise à obtenir un stage dans votre entreprise.

## Commencement HTML

Au départ, j'ai créé la structure HTML selon ma vision, mais j'ai rencontré des difficultés, notamment avec l'emplacement des photos. Après quelques ajustements, j'ai choisi une approche en utilisant une grille CSS avec deux colonnes différentes, facilitant la gestion des éléments.

*Le code HTML initial est en commentaire dans le fichier index.html*

## Style CSS (général)

La réalisation du CSS a été complexe mais agréable. Malgré des erreurs, j'ai résolu les problèmes grâce à des recherches en ligne et mes connaissances antérieures.

## Toggle Switch en CSS

La création du Toggle Switch en CSS a été un défi. Après avoir essayé de le concevoir moi-même sans succès, j'ai trouvé un template en ligne que j'ai adapté pour obtenir le résultat souhaité, bien que légèrement différent de la maquette.

## DarkMode JS

La mise en place du Dark Mode avec JavaScript s'est déroulée sans difficulté. Pour intégrer les préférences du navigateur, j'ai adapté un code trouvé en ligne, en le décomposant pour comprendre son fonctionnement.

## Le Neumorphisme

La réalisation du neumorphisme a été plus complexe que prévu. Après avoir cherché des templates de box shadows, j'ai adapté leur utilisation pour obtenir un rendu similaire à la maquette.

## Le Responsive

La partie responsive pour les téléphones n'a pas posé de problème majeur. Cependant, des ajustements ont été nécessaires pour les grands écrans. J'ai créé ma propre méthode en utilisant l'inspecteur Chrome pour vérifier le rendu sur une variété de tailles d'écran, du 4K aux 13 pouces.

## Serveur HTTP

Malheureusement, je n'ai pas eu le temps de mettre en place le serveur HTTP. J'ai exploré des vidéos sur YouTube pour comprendre son fonctionnement avec Node.js et Express.

## Utilisation

La maquette a été créée avec Visual Studio Code et sa fonctionnalité Live Server.

## Conclusion

Merci d'avoir pris le temps de lire ce README. Ce projet, complémentaire à mes cours, a été enrichissant. Contrairement aux exercices scolaires, j'ai cherché à m'impliquer davantage pour obtenir un rendu professionnel, et j'ai beaucoup appris grâce à cette opportunité.

Mathis Petit, Apprenant Simplon.
